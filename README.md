Multiplex single win for a group of term buf.

## highlight
* Togglable layout.
* Debugprint extmarks.
* Navigation in debugprint lines.

```sh
nvim --clean --cmd 'set rtp^=.'
```

> [!NOTE]
> Enhancement of terminal mode/cursor: https://github.com/phanen/termmode.nvim.
